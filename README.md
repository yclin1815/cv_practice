# 單頁式網頁 模擬CV

* Demo: [https://yclin1815.github.io/cv_practice/](https://yclin1815.github.io/cv_practice/)

## 簡介

* Bootstrap、Flex 網頁切版
* CSS keyframes 互動按鈕
* 全頁式導覽效果
* 視差滾動效果
* 元素進場動畫效果
* 圖片放大燈箱效果

## 使用技術

* HTML + CSS
* Bootstrap
* jQuery
* jQuery 插件
* RWD
